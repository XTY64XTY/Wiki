# Xdows Tools

## Introduction

Xdows Tools is the built-in toolset of Xdows Security, providing system management and security auxiliary functions.

## Tool List

### Process Manager

Manage running processes in the system:

- **View Process List**: Display process name, PID, thread count, handle count, and memory usage
- **Sort Function**: Support sorting by name, PID, memory, thread count, handle count
- **View Switching**: Support list view and tree view
- **Search Process**: Support searching by process name or PID
- **End Process**: Terminate selected process
- **Suspend/Resume Process**: Pause or resume process execution
- **View Details**: Double-click process to view detailed information

### Command Prompt

Built-in terminal tool:

- **Execute Commands**: Input and execute system commands
- **Command History**: Support browsing history with up/down arrow keys
- **Clear Output**: One-click clear terminal output
- **Copy Output**: Copy terminal content to clipboard
- **Restart Terminal**: Restart command prompt session

### Popup Blocker

Custom rules to block popups:

- **Add Rule**: Add blocking rules based on window title and process name
- **Edit Rule**: Modify existing blocking rules
- **Delete Rule**: Remove unwanted rules
- **Enable/Disable Rule**: Control individual rule switches
- **Master Switch**: One-click enable or disable popup blocking
- **Rule Status**: Display rule match count and current status
