# Xdows Tools

## 简介

Xdows Tools 是 Xdows Security 内置的工具集，提供系统管理和安全辅助功能。

## 工具列表

### 进程管理器

管理系统中运行的进程：

- **查看进程列表**：显示进程名称、PID、线程数、句柄数和内存占用
- **排序功能**：支持按名称、PID、内存、线程数、句柄数排序
- **视图切换**：支持列表视图和树状图视图
- **搜索进程**：支持按进程名称或 PID 搜索
- **结束进程**：终止选中的进程
- **挂起/恢复进程**：暂停或恢复进程运行
- **查看详细信息**：双击进程查看详细信息

### 命令提示符

内置终端工具：

- **执行命令**：输入并执行系统命令
- **命令历史**：支持使用上下方向键浏览历史命令
- **清空输出**：一键清空终端输出
- **复制输出**：复制终端内容到剪贴板
- **重启终端**：重新启动命令提示符会话

### 弹窗拦截器

自定义规则拦截弹窗：

- **添加规则**：根据窗口标题和进程名添加拦截规则
- **编辑规则**：修改现有拦截规则
- **删除规则**：移除不需要的规则
- **启用/禁用规则**：单独控制每个规则的开关状态
- **主开关**：一键开启或关闭弹窗拦截功能
- **规则状态**：显示规则的匹配次数和当前状态
