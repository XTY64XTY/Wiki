## 核心 {#Kernel}

Xdows Security 的 3.0 和 4.0 都基於網頁技術構建 UI ，但其與UI互動的方式有所不同

 - 3.0 版本只是讓瀏覽器載入網頁然後核心捕獲網頁的 js 操作達到目的，並且除了 WebUI 仍然有其它組件，這些組件不可自訂

 - 4.0 版本是真正的可以和使用者瀏覽器互相發送資料，並且只有 WebUI，大大提高了可自訂性，同時UI還有一定的跨平台性
