# 外掛

## 介紹

外掛系統旨在為 Xdows Tools 提供輕巧簡潔的擴充功能。

## 差異

外掛系統源自 Xdows Security 3.01。

當時，外掛系統整合在主程式中，而不是 Xdows Tools 中。

在 3.0 版本中，外掛本質上是 DLL 檔案，主程式會在某些操作期間呼叫 DLL 以實現外掛功能。

在 Xdows Security 3.03 中，外掛系統已更新，支援使用外掛作為掃描引擎。

在 Xdows Security 3.04 中，由於各種原因，外掛系統**已被移除**。

在 Xdows Security 4.00 中，外掛系統已重新引入並整合到 Xdows Tools 中。

## 外掛

4.0 版本中的外掛系統整合到 Xdows Tools 中，而不是主程式中。

這意味著它**失去**了對主程式的某些存取權限，但提高了安全性和穩定性。

外掛本質上是一個包含設定檔、程式檔案和輔助網頁檔案的目錄。

該目錄通常包含以下內容：

 - `Plugins` 目錄，用於儲存外掛本身，通常處於**不可編輯**狀態。
 - `Data` 目錄，用於儲存外掛所需的資料。

## 外掛套件

外掛套件是包含一個或多個外掛的 ZIP 壓縮檔。

您可以透過 `Xdows Tools` - `匯入外掛` 匯入外掛套件。
