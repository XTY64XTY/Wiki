# Xdows Tools

## 簡介

Xdows Tools 是 Xdows Security 內建的工具集，提供系統管理和安全輔助功能。

## 工具列表

### 進程管理器

管理系統中執行的進程：

- **檢視進程列表**：顯示進程名稱、PID、執行緒數、控制代碼數和記憶體佔用
- **排序功能**：支援按名稱、PID、記憶體、執行緒數、控制代碼數排序
- **檢視切換**：支援列表檢視和樹狀圖檢視
- **搜尋進程**：支援按進程名稱或 PID 搜尋
- **結束進程**：終止選中的進程
- **暫停/恢復進程**：暫停或恢復進程執行
- **檢視詳細資訊**：雙擊進程檢視詳細資訊

### 命令提示字元

內建終端機工具：

- **執行命令**：輸入並執行系統命令
- **命令歷史**：支援使用上下方向鍵瀏覽歷史命令
- **清空輸出**：一鍵清空終端機輸出
- **複製輸出**：複製終端機內容到剪貼簿
- **重新啟動終端機**：重新啟動命令提示字元工作階段

### 彈窗攔截器

自訂規則攔截彈窗：

- **新增規則**：根據視窗標題和處理程序名稱新增攔截規則
- **編輯規則**：修改現有攔截規則
- **刪除規則**：移除不需要的規則
- **啟用/停用規則**：單獨控制每個規則的開關狀態
- **主開關**：一鍵開啟或關閉彈窗攔截功能
- **規則狀態**：顯示規則的匹配次數和目前狀態
